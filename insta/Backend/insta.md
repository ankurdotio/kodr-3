# Authentication


1. [POST] /api/auth/register/  - Register a new user
2. [POST] /api/auth/login/  - Login and obtain JWT token
3. [POST] /api/auth/logout/  - Logout and invalidate JWT token
4. [GET] /api/auth/me - Get current authenticated user's profile
5. [GET] /api/auth/google - Initiate Google OAuth2 login
